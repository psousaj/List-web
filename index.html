    FIFO:
    A estrutura de dados fila segue um padrão conhecido como FIFO (first-in first-out), onde o primeiro a entrar é o primeiro a sair. Imagine o exemplo em uma fila de banco

    FILO:
    LIFO(acrônimo para a expressão inglesa Last In, First Out) É equivalente a FILO, que significa First In, Last Out. Refere-se a estrutura pilha. Uma pilha é uma lista linear na qual o primeiro elemento a entrar é o último elemento a sair. Ela possui apenas uma entrada, chamada de topo, a partir daqual os dados entram e saem dela
